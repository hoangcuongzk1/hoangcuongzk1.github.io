# Hiểu Về React Hooks

React Hooks đã cách mạng hóa cách chúng ta viết component.

## useState

`const [count, setCount] = useState(0);`

Đây là hook cơ bản nhất để quản lý state.

## useEffect

Được sử dụng cho các side effects như gọi API. Hãy nhớ xử lý dependency array cẩn thận!
