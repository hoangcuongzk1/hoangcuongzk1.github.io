# TypeScript là gì?

TypeScript là một siêu tập hợp (superset) của JavaScript. Nó thêm vào static typing.

## Lợi ích

1. **Gợi ý code tốt hơn**: Intellisense hoạt động tuyệt vời.
2. **Ít lỗi hơn**: Bắt lỗi ngay khi gõ code.
3. **Dễ bảo trì**: Code rõ ràng, dễ hiểu hơn.

```js
const greeter = (name: string) => {
  return `Hello ${name}`;
}
```
