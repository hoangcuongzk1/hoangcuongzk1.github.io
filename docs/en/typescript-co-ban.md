# What is TypeScript?

TypeScript is a superset of JavaScript. It adds static typing.

## Benefits

1. **Better Intellisense**: Code completion works great.
2. **Fewer Bugs**: Catch errors while typing.
3. **Maintainability**: Code is clearer and easier to understand.

```js
const greeter = (name: string) => {
  return `Hello ${name}`;
}
```
